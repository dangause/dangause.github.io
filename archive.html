Predicting NCAA March Madness Games with Machine Learning

This is a project for my Spring 2020 Statistical Learning class at Middlebury. In this project, I gathered an cleaned large datasets, then built models using LCA and GLM algorithms to predict the outcomes of NCAA March Madness mens basketball games.
